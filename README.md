# TON 红包 Bot

一个基于 Telegram 的 TON 区块链红包机器人，支持在群组中发送和领取红包。

## 功能特性

### 🎁 红包功能
- **发红包**: 在群组中发送 TON 红包
- **领红包**: 群成员一键领取红包
- **两种模式**: 拼手气红包 / 普通红包
- **防重复**: 每人只能领取一次
- **自动过期**: 24小时后自动过期

### 💰 钱包功能
- **余额查询**: 实时查看账户余额
- **充值**: 向Bot钱包转账充值
- **提现**: 提取余额到个人钱包
- **交易记录**: 查看详细交易历史

### 🔒 安全特性
- **托管钱包**: Bot代为管理用户资产
- **交易记录**: 完整的资金流水
- **余额保护**: 防止超额支出
- **地址验证**: 严格的地址格式检查

## 部署指南

### Docker 部署

```bash
# 1. 克隆项目
git clone <repository-url>
cd PocketBang_bot

# 2. 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填入必要的配置信息

# 3. 启动服务
docker-compose up -d --build

# 4. 查看日志
docker-compose logs -f

# 5. 停止服务
docker-compose down
```
